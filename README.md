A fully custom, multi-page portfolio site built as a single-page application — no frameworks, just hand-rolled HTML, CSS, and vanilla JavaScript. The site is a sidebar-navigated SPA with seven sections: Home, About, Projects, Experience, Skills, Research, and Contact.

### ✨ Highlights

- **Zero dependencies** — pure HTML/CSS/JS, no build step required
- **Smooth page transitions** with custom reveal animations and easing curves
- **Fully responsive** — collapses gracefully down to mobile breakpoints
- **Dynamic skill bars** that animate on section entry
- **Dark-themed code terminal** on the home page, styled like a real IDE
